Libraries.io https://libraries.io

Libraries.io monitors 815,876 open source libraries across 29 different package managers written in over 160 different languges.

You can discover new libraries to use in your software projects as well as be notified of new releases to keep your applications secure and up to date.

Libraries.io aims to help both maintainers and users of open source software with a range of tools to improve visibility across changes in the ecosystem.

Follow @librariesio

Features


  • Powerful search using a PageRank style algorithm to suggest the most dependable software, filterable by language, platform, license and keyword.
  • Close to realtime updates of new libraries and new versions of existing libraries, usually in sync within 10 minutes of new releases.
  • Subscribe to email notifications of new versions of libraries that you depend upon.
  • Open source license normalization with all valid SPDX licenses to help maintain compliance.
  • Standardized RSS feeds and JSON APIs across all platforms to work with the open web including a realtime Firehose API.
  • GitHub repository dependency parsing technology, indexing the dependency graph of the whole open source ecosystem.

Coming soon


  • Automatic dependency notifications for repositories, keeping track of dependencies you use and notifying you of updates automatically as you build your app.
  • Dependency based Library recommendation engine, suggesting open source projects based on the libraries you actually use in your code.
  • Web hooks and integrations with many popular services including Slack, Hipchat, Issue trackers and more.
  • Security advisory notifications - Get notified of security releases and affected dependent projects the instant they are announced.
  • Curated collections of libraries - User generated groups of libraries, hand picked to solve particular problems.
  • Automatic changelog generation - see all relevant changes to a library when new versions are released, even if the author doesn't maintain a changelog.
